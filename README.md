# fixabay

Pixabay API를 이용하여 만든 무료 이미지 검색 사이트

## 메인 스택

- React
- Typescript
- SWR / ReactQuery
- tailwindcss
- styled-components
- recoil
- i18next

## 참고

SWR과 ReactQuery를 비교하기 위해서 2가지 버전으로 페이지를 만들었습니다.

배포버전에는 ReactQuery를 사용한 경로로 redirect 됩니다.
